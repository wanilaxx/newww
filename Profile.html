สวัสดีค่ะ ดิฉันนางสาววริยา เข็มทองคำ ชื่อเล่น หยก ดิฉันตั้งใจสร้างเว็บไซต์นี้เพื่อแสดงทักษะในการใช้ HTML CSS เบื้องต้นและแสดงผลงานของดิฉันเพิ่มเติมที่ดิฉันได้ไปแข่งขันทางวิชาการต่างๆตลอดทั้งช่วงเวลามัธยมศึกษาตอนปลายนี้ หวังว่าคณะกรรมการจะมองเห็นถึงความตั้งใจของดิฉันได้มากยิ่งขึ้น ในการสร้างเว็บไซต์นี้ขึ้นมาเกิดจากความสนใจของฉันในด้านคอมพิวเตอร์ จึงได้ศึกษาหาความรู้เพิ่มเติมด้วยการสมัครเรียนออนไลน์ในช่วง ปิดเทอมและศึกษาจากแหล่งความรู้อื่นๆจนเกิดขึ้นมาเป็นเว็บไซต์นี้ ขอบคุณค่ะ
